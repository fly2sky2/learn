# learn

### 一个学习强国的辅助(挑战答题、双人对战、四人赛)脚本，稳定使用3年


#### 软件需要几个权限
1. SD卡的存储权限，用来存放题库。
2. 悬浮窗权限，用来展示答案。
3. 无障碍权限，用来识别答案。


#### 操作步骤
1. 首先打开软件，给予对应的权限，然后点击开启图标,（打开无障碍后，还需要手动点击开启图标，直到界面显示，正在识别题目中～）
2. 打开学习强国软件，点击挑战答题。四人赛、对战答题后，软件会自动答题。(0078)
